                                                            //********** EASY **********\\

1- Verificar se um número é primo
Crie uma função que verifique se um número fornecido é primo. Um número primo é aquele que só é divisível por 1 e por ele mesmo.


2- Calcular o fatorial de um número
Crie uma função que calcule o fatorial de um número. O fatorial de um número n é o produto de todos os números inteiros de 1 até n.


3- Contar vogais em uma string
Crie uma função que conte o número de vogais em uma string fornecida. Considere as vogais 'a', 'e', 'i', 'o', 'u' tanto maiúsculas quanto minúsculas.


4- Calcular a média de um array de números
Crie uma função que receba um array de números e retorne a média desses números.


5- Remover itens duplicados de um array
Crie uma função que receba um array e retorne um novo array sem itens duplicados.


6- Contar palavras em uma string
Crie uma função que conte o número de palavras em uma string. Considere que as palavras estão separadas por espaços.


7- Verificar se uma string é um palíndromo
Crie uma função que verifique se uma string fornecida é um palíndromo. Um palíndromo é uma palavra, frase ou número que permanece igual quando lido de trás para frente, desconsiderando os espaços.


8- Calcular a soma de números em um intervalo
Crie uma função que receba dois números e retorne a soma de todos os números inteiros entre esses dois números, incluindo os próprios.


9- Inverter uma string
Crie uma função que inverta a ordem dos caracteres de uma string fornecida.


10- Encontrar o maior número em um array
Crie uma função que receba um array de números e retorne o maior número desse array.

1- Verificar anagramas
Crie uma função que verifique se duas strings fornecidas são anagramas uma da outra. Duas strings são anagramas se podem ser rearranjadas para formar a outra.


2- Calcular a soma dos dígitos de um número
Crie uma função que receba um número e retorne a soma dos seus dígitos.


3- Encontrar a sequência crescente mais longa em um array
Crie uma função que encontre a sequência crescente mais longa em um array de números.


4- Verificar se uma matriz é simétrica
Crie uma função que verifique se uma matriz quadrada é simétrica. Uma matriz é simétrica se é igual à sua transposta.


5- Converter número romano para número inteiro
Crie uma função que converta um número romano (fornecido como string) em um número inteiro.


6- Mover zeros para o final do array
Crie uma função que mova todos os zeros de um array para o final, mantendo a ordem dos outros elementos.


7- Encontrar números duplicados em um array
Crie uma função que encontre todos os números duplicados em um array e os retorne em um novo array.


8- Remover elementos falsy de um array
Crie uma função que remova todos os elementos falsy (false, null, 0, "", undefined, NaN) de um array.


9- Encontrar o menor número ausente em um array
Crie uma função que encontre o menor número positivo que está ausente em um array.


10- Encontrar a interseção de dois arrays
Crie uma função que receba dois arrays e retorne um novo array contendo apenas os elementos que aparecem em ambos os arrays.
